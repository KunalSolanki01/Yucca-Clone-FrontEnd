# Yucca Packaging Website Clone

A **front-end clone of the Yucca Packaging website** built using **HTML, CSS, and JavaScript**.
This project recreates the UI, layout, animations, and interactive sections of the original Yucca website to practice modern front-end development.

The real Yucca Packaging company provides **innovative and sustainable food packaging solutions for food service, food processing, and agriculture industries**. ([Yucca Packaging][1])

# 📌 Project Overview

This project replicates the **homepage design and animations** of the Yucca Packaging website.

The goal of this project was to practice:

* UI cloning
* Advanced CSS animations
* Hover interactions
* Scroll effects
* Modern website layout structure

---

# 🚀 Features

## Navigation Header

* Fixed header
* Hover underline animation
* Cart and profile hover effects
* Smooth UI transitions

## Hero Section

* Full-screen hero image
* Overlay heading text
* Animated glassmorphism cards

## Industry Cards

Includes three sections:

* Food Service
* Food Processing
* Agriculture

Each card includes:

* Animated text reveal
* Hover transitions
* Arrow animations

## Marquee Section

Scrolling text banners using CSS animations.

## Product Section

New products showcase with:

* Hover zoom effects
* Quantity buttons
* Product price display

## Custom Solutions Section

Information about packaging customization with animated scrolling text.

## Certifications Section

Displays packaging compliance standards and certifications.

## FAQ Section

Interactive FAQ layout.

## Footer

* Social media icons
* Contact links
* Copyright information

# 📦 Libraries Used

## Locomotive Scroll

Used to implement **smooth scrolling and scroll-based effects**.

CDN used:

```
https://cdn.jsdelivr.net/npm/locomotive-scroll/bundled/locomotive-scroll.min.js
```

Initialization:

```javascript
const locomotiveScroll = new LocomotiveScroll();
```

---

## Font Awesome

Used for icons.

CDN:

```
https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css
```

# ⚙️ Installation

Clone the repository

```
git clone https://github.com/KunalSolanki01/Yucca-Clone-FrontEnd.git
```

Open the project folder

```
cd Yucca-Clone-FrontEnd
```

Run the project by opening **index.html** in your browser.
